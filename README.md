# Aura Boutique – Small‑Scale E‑commerce MVP

A production‑ready, mobile‑first e‑commerce starter for a boutique store selling jewellery, sarees, and ladies' & girls' clothing. Built with modern tools, focused on speed, trust, and maintainability.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Database**: PostgreSQL + Prisma ORM
- **Auth**: NextAuth (Auth.js) with credentials
- **Payments**: Stripe Checkout
- **Validation**: Zod
- **State**: Zustand (cart) + React Context where needed
- **Deployment**: Vercel‑ready

## Getting Started

### Prerequisites

- Node.js 20+
- PostgreSQL (local or cloud)
